SIMION ASMS Courses

The following are course notes from previous SIMION courses offered at the ASMS conferences. Corresponding SIMION data files are also included under the SIMION "courses" subdirectory.

Note: These courses are a bit dated (SIMION 7+6), but they are still relevant. The SIMION data files can be used unmodified under SIMION 8, but the SIMION 8 user interface differs a bit (but still is conceptually similar). Some of the course sections deal with PRG user programming, which can still be used in SIMION 8, even though SIMION 8 Lua programming is now preferred.
