The files in this directory provide an example of a simple einzel lens.
 
 
einzel.iob        Einzel demonstration file
 
 
How to run demo:
 
    1.  View einzel.iob
    2.  Load its associated .FLY file
    3.  Set computational quality to 0
    4.  Fly Ions grouped in rerun mode
    5.  Notice point of focus
    6.  Click PA Adj button and then click Fadj button while ions are flying
        change electrode 2 to 150 volts and see the change in focus point
    7.  Switch to PE view and click the Dots button now you can see why
        the ions focus
    8.  Exit PE view and stop flying ions (click the fly'm button)
    9.  Click the button just the the right of Grouped until it shows
        Beam Repulsion.
    10. Fly the ions -  Increase the beam current to 1.0e-7 amps
        Notice the effect this has on ion trajectories
         
    Note:  This is a good demo to use to learn about 3D zooming and other
           basic SIMION 6.0 features.  Read your manual, spend some time
           and enjoy!
