To prepare this Quad demo:

	1. Remove all PAs from RAM
	2. Use New on the QUAD.GEM FILE to create a potential array.
	3. Save the potential array as QUAD.PA#
	4. Refine the potential array

	5. Remove all PAs from RAM
	6. Use New on the QUADIN.GEM FILE to create a potential array.
	7. Save the potential array as QUADIN.PA#
	8. Refine the potential array

	9. Remove all PAs from RAM
   10. Use New on the QUADOUT.GEM FILE to create a potential array.
   11. Save the potential array as QUADOUT.PA#
   12. Refine the potential array

   13. Remove all PAs from RAM
   14. Load QUAD.IOB with View
