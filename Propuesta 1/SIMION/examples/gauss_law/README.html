SIMION Gauss's Law Charge/Capacitance and Field Energy Calculation Example

Purpose

This example demonstrates methods of calculating the total charge (or the capacitance) inside a given volume via Gauss's Law[1] or via the total field energy. It does so by evaluating integrals over the SIMION calculated electric field via the SL Libraries for Perl or the Lua simion.pas and simionx.FieldAnalysis libraries.

NOTE: This example requires knowledge of Perl or Lua programming languages.

Description

The total charge inside a given volume Ω bounded by some surface S can be computed from Gauss's Law[1] if the electric field E over that surface is known (which you can calculate in SIMION):

charge = ε0 * ∫∫S E * dA

An alternate method, for a capacitor, is to first compute the energy W of the electric field E inside a volume containing both electrodes:

W = ∫∫∫Ω (0.5*ε0) E2 dτ

The energy W can be related to the charge Q on the two electrodes via the voltage difference ΔV as W = QΔV/2.

So, the steps required to compute total charge inside a volume are

  • (1) Create and refine a SIMION potential array (solving the fields).
  • (2) Compute the appropriate integral over some region of the potential array to find total charge. Optionally use that to compute a capacitance[2]: C = Q / |ΔV|.

Computing the integral can be done in various ways. You might even utilize SIMION Data Recording. However, this task is best facilitated instead with a custom program contained here. This program will examine a SIMION potential array (PA) file and calculate the total charge in the specified sphere or box surface containing an electrode. The integration is done using a fairly simple but effective quasirandom Monte-Carlo method.

As an example, we'll use here the simple spherical capacitor system consisting of two concentric spherical electrodes with radii R1=80 mm and R2=120 mm and voltages V1=500 V and V2=-1000/3 V. According to theory, the charge on the inner sphere should have charge Q = (ΔV) * 4 π ε0 (R1-1 - R2-1)-1, giving Q ~= 2.225301E-8 C, or the negative of that on the outer sphere. The voltage along r (in m) should be V=200000*(1/r) - 2000, so the field should be E = 200000r-2. The energy stored in the capacitor is (1/2)CV2 = (1/2)QV = 9.272084E-6 J.

There are two versions of the program: Perl and Lua. The Perl version uses the SL Libraries for Perl. The Lua version uses the Lua simion.pas and simionx.FieldAnalysis libraries in SIMION 8.1. The Perl version was developed earlier and has fewer capabilities than the Lua version.

Files in this demo:

  • README.html - Description of example
  • Perl version:
    • run.pl - Perl batch mode program to do the calculation (requires a Perl interpreter be installed)
    • FieldAnalysis.pm - Perl module containing the integration routines
  • Lua version:
    • run.lua - Lua batch mode program to do the calculation (requires SIMION 8.1)
  • sc2d.gem, sc2d10.gem, and sc3d.gem - various GEM file models of spherical capacitors, in 2D and 3D, including a 2D one at 10x resolution.
  • sc2d.pa, sc2d10.pa, and sc3d.pa - PA files you must generate from the GEM files.

Usage - Perl Example

How to run the demo:

  • 1. Ensure you have Perl installed (this was tested on Perl 5.8).
  • 2. Convert one of the GEM files to a PA, refine in SIMION, and save.
  • 3. In the run.pl program edit the $pa_name and $lambda variables to match the PA file you created.
  • 4. Run the run.pl Perl program. That, is from the command-line in the directory of this project run "perl run.pl".
  • 5. The solution will converge and display on the screen.

For this spherical capacitor system, the run.pl program is configured basically as follows: 

 # $pa_name: location of the SIMION potential array (PA) file.
 # $L: mm/gu scaling factor used in the PA.
 my ($pa_name, $L) = ('sc3d.pa', 1);
 
 # load potential array
 my $pa = SIMION::PA->new(file => $pa_name);  
 
 # load potential array
 my $pa = SIMION::PA->new(file => $pa_name);

 # calculate total charge using Gauss's Law.
 print "EXPECTING: 2.225301E-8\n";
 &FieldAnalysis::charge_from_gauss_law_display(
   $pa,
   &FieldAnalysis::sphere(0,0,0, 100),
   $L,
   0.1, 0.0001
 );

SIMION results are 2.249E-8 C (3D), 2.242E-8 C (2D), 2.2269E-8 C (2D, lambda=10 gu/mm).

Usage - Lua Example

The Lua version has a number of advantages such as the ability to run it directly inside SIMION (the implementation is also newer and more complete).

This example uses SIMION 8.1 features.

How to run the demo:

  • 1. Click "Run Lua Program" from the SIMION main screen and select the "run.lua" program. Output is displayed on the log window.

Further details are available in the documentation on the Lua simionx.FieldAnalysis library.
