SIMION™ 8.1

SIMION Miscellaneous Examples

Purpose

The files in this directory various miscellaneous examples of potential arrays and other things.

Files in this demo:

  • README.html - Description of examples
  • Extract
    • extract.iob - SIMION workbench for extraction system
    • extract.pa - SIMION potential array of extraction lens for extract.iob
    • extract.fly - SIMION particle definitions for extract.iob
    • extract.con - SIMION contour definitions for extract.iob
  • Point
    • point.pa - SIMION potential array of point lens.
    • point.con - SIMION contour definitions for point.pa.
  • Test 3D
    • test3d.iob - SIMION workbench of 3D example
    • test3d.pa - SIMION potential array of 3D example for test3d.iob.
    • test3d.fly - SIMION particle definitions for test3d.iob.
  • Test Cylindrical
    • testcyl.iob - SIMION workbench of 2D cylindrical lens
    • testcyl.pa# - SIMION potential array of 2D cylindrical lens for testcyl.iob.
    • testcyl.fly - SIMION particle definitions for testcyl.iob (line of ions)
    • testcyl.con - SIMION contours definitions for testcyl.iob.
    • beam.fly - SIMION particle definitions for testcyl.iob (area beam of ions)
  • Test Sym
    • testsym.pa - SIMION potential array - 2D planar example
  • Gas flow import
    • fluent_ip_to_simion_pas.lua - convert gas flow in Fluent .ip file to SIMION PA's.
    • fluent_example.ip - trivial example Fluent .ip file used by fluent_ip_to_simion_pas.lua
    • comsol_to_simion_pas.lua - convert COMSOL gas flow files (comsol_vx.txt and comsol_vy.txt) to SIMION PA's (8.2 early access)
  • Other
    • particle_distance_traveled.lua - user program that can compute total distance (in mm) a particle has traveled. Attached this to a workbench to use it.
    • first_uniqueness.gem/first_uniqueness_cage.gem - Used to demonstrate first uniqueness theorem. See Doc(First Uniqueness Theorem)

Usage

How to run each demo:

  • 1. View extract.iob, test3d.iob, or testcyl.iob. (Click View on main screen and select the IOB file you want.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Fly ions. (Click Fly'm button.)
  • 4. Experiment.

Usage of fluent_ip_to_simion_pas.lua:

This is an example of converting a gas flow computed by Fluent into SIMION PA format. See Doc(CFD) for background. The "fluent_ip_to_simion_pas.lua" can be run from the "Run Lua Program" button on the SIMION main screen. However, you should first edit this file according to your needs though.

Usage of comsol_to_simion_pas.lua (version 8.2 early access):

This is an example of converting a gas flow computed by COMSOL® into SIMION PA format. See Doc(CFD) for background. The "comsol_to_simion_pas.lua" can be run from the "Run Lua Program" button on the SIMION main screen. However, you should first edit this file according to your needs though.
