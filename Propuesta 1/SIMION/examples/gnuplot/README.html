SIMION gnuplot Examples

Purpose

These examples demonstrate how to invoke gnuplot from SIMION user programs. gnuplot is a popular free third-party graphing program. Using gnuplot is an alternative to using Excel, which unlike gnuplot is not available under Linux, and you may prefer gnuplot even under Windows.

Please note that if you want your code to also be usable with other plotting programs besides gnuplot (e.g. Excel), then it is recommended to instead use the generic plotting library available in Example(plot). This generic library provides a standard interface for working with both the Excel and gnuplot libraries.

Installing gnuplot

If you're running SIMION under Windows, you may download an SIS build of the gnuplot program from the "Check for Updates" button in SIMION 8.1 and place the contained "gnuplot" folder inside you SIMION program folder (i.e. the gnuplot executable would typically be located at "c:\Program Files\SIMION-8.1\gnuplot\bin\gnuplot.exe"). You may alternately use your own build of gnuplot (see downloading gnuplot), or you may install Cygwin (which is a Unix-like environment that runs under Windows and provides an easy way to deploy Unit-style utilities on Windows) and select the "gnuplot" package during setup. If you're running SIMION under Linux via Wine, then just install the gnuplot package in your distribution, typically via "sudo apt-get install gnuplot" if doing this from the command-line, or you may run the SIS build (Windows version) of gnuplot under Wine by installing it the same way you do under Windows.

Files in this demo:

  • README.html - Description of example
  • gnuplotlib.lua - Library providing a simple interface to GNU plot.
  • gnuplotlib_example.lua - simple batch mode program to test gnuplotlib.lua.
  • direct.lua - batch mode program that executes gnuplot without gnuplotlib.lua.
  • simple.gp - trivial example that can be run directly in gnuplot (a simple test)
  • simple_cygwin.gp - similar to simple.gp but sets the terminal appropriately for Cygwin.

Usage

Usage is described below. Note: before running the examples, ensure gnuplot is setup on your system as described above.

  • 1. Run gnuplotlib_example.lua. (Click Run Lua Program on main screen and select the gnuplotlib_example.lua.) Some plots will display on your screen. You may alternately try running the direct.lua example. Examine these Lua files in a text editor to see how they work.

Further Notes

The SIS build of gnuplot contains a Windows help file, wgnuplot.chm, describing the gnuplot commands. Documentation is also available on the web page http://www.gnuplot.info/ (see Documentation and Screenshots links). For a quick view of many different gnuplot plotting features, load the "demo\all.dem" example in the wgnuplot.exe program provided in the gnuplot distribution.
