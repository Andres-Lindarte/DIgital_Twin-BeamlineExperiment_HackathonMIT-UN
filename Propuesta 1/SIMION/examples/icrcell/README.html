SIMION Ion Cyclotron Resonance (ICR) Example

Purpose

The files in this directory provide an example of having user programs model an ICR cell. This is a relatively complete 3D model of an ICR cell.

Description

This demo creates 20 random ions to the left of the ICR cell. These ions enter the cell and are decelerated by a buncher type field. When they are around the middle of the trap the RF sweep starts (ions turn red). After The RF sweep finished the ions turn green and the encaps are set to trap the ions. The PE Surface views of the Z=0 plane can be used to view the various voltage forcing functions (PE surfaces are updated).

A dummy 2D planar magnetic potential array - mag.pa is used to create the magnetic field. mag.pa provides no field itself. However the user program file mag.lua has a mfield_adjust program segment that defines the magnetic field from the user adjustable variable bx_gauss (30000 default).

RAM requirements are aroung 20 megs total.

Note: Ions are generated randomly in energy, cone angle, 3D offset, and time. You have adjustable variables to control each aspect.

Files in this Demo:

  • README.html - description of this example
  • icr.iob - workbench of ICR
    • icr.fly2 - particle definitions, with randomized starting conditions
    • icr.pa# - icr cell, definition PA (refines to icr.pa0)
    • mag.pa - empty magnetic PA to hold magnetic field defined by icr.lua.
    • icr.lua - workbench user program to apply RF voltages and B-field

Usage

How to run ICR demo:

  • 1. View icr.iob. (Click View on main screen and select the icr.iob file.) (If this is your first time using the example, you will be asked to refine (confirm that and press Refine).) Notice that the Particles tab has Grouped enabled and T.Qual = 0.
  • 2. Fly ions Dots that are fast. (In Particles tab, enable Dots, and move slider below that to the right.) Ions start outside the ICR cell (ions - blue), enter the cell, decelerate, RF sweep starts (ions - red), ions accelerate, and RF sweep stops (ions - green).
  • 3. View ions in side, top, and end 2D views (as ions fly). Cut top off and front of ICR cell and use isometric views to look inside. (See printed manual for help on 3D zooming features, or skip this.)
  • 4. Switch from Dots back to lines as the ions fly. (In Particles tab, disable Dots.) Notice the interesting patterns with lines.
  • 5. Cut off the left half of the cell in the ZY view (3D zoom). Switch to XY view and fly ions Grouped as Dots in PE surface view. Notice the initial deceleration field, the RF pumping, and the final traping end fields.
  • 6. Experiment on the effects of ion initial conditions, etc. This model provides a lot of insight into why ICRs work like they do if you spend some time with it.

Note: You can modify the user programs and PAs to reflect real world ICRs. This demo serves as a good starting point for ICR models.
