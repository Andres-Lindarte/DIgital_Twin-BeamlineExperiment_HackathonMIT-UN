SIMION Particle Randomization Example

Purpose

The files in this directory provide an example of having user programs create random ions. A simple initialize program segment is used to randomize the energy and directions of all ions flown.

Description

The program has two adjustable variables:

  • percent_energy_variation sets the the range of energies to use in terms of the percentage of the ion's initially defined energy. E.g. 50 means +- 50% of defined energy. For a 0.1 eV ion the range of energies would be 0.05 to 0.15 eV.
  • cone_half_angle sets the range of azimuth and elevation angle changes in degrees. E.g. 90 means +- 90 degrees. For an ion with az = el = 0 degrees, the az and el angles would be +- 90 degrees (half sphere).

The program sets initial ion direction down x-axis. It rotates elevation -90 to make y-axis the initial beam axis. This provides azimuth angle symmetry around beam axis (better symmetry for direction randomization). Transform is reversed before returing velocity components.

This should serve as a simple example of how random ion generation problems might be approached.

Note: Rather than use a program as done in this example, a similar effect can often be achieved more easilly with FLY2 files (new SIMION 8 feature--see FLY2 appendix in printed SIMION manual) via the cone distribution. This example originated from SIMION 7, which did not have FLY2 files, but the approach given can still be useful at times.

Usage

How to run demo:

  • 1. View random.iob. (Click View on main screen and select the random.iob file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Set trajectory quality to 0. (In Particles tab, set TQual to 0.)
  • 4. Fly ions Grouped (lines or Dots). (In Particles tab, enable Grouped and optionally enable Dots. Click Fly'm button.)
  • 5. Use PE view to get a better understanding of how random ions fly. (Press PE button.)
  • 6. Enable Rerun and changes variables to see effect. (In Particles tab, enable Rerun. In Variables tab, change one of the variables as desired and press Enter.)
