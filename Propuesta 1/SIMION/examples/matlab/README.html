SIMION™ 8.1

SIMION Examples for MATLAB/Octave

Purpose

These are examples of using SIMION® with MATLAB® or Octave programs.

Note: These example files are only available SIMION 8.2 early access.

Files in this Example

  • simion_pa_save.m - MATLAB function to save a MATLAB array to SIMION 8.1 PA file.
  • simion_pa_save_example.m - MATLAB script example of using simion_pa_save.m.
  • simion_call_example.m - MATLAB script example of calling SIMION from MATLAB via the command-line interface.
  • matlab_call_example.lua - SIMION Lua script example of calling MATLAB from SIMION via the command-line interface.
  • matlab_com_example.lua - SIMION Lua script example of calling MATLAB from SIMION via the COM automation API.
