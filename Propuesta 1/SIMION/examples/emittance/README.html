SIMION™ 8.1

SIMION Emittance Calculation Example

Purpose

Calculates emittance of beam.

Description

This example uses a user program to calculate the emittance of beam at the splat location.

For general background on emittance, see Emittance.

Files in this demo:

  • README.html - Description of example
  • emittance.iob - SIMION workbench
  • emittance.fly - SIMION particle definitions associated with emittance.iob
  • emittance.pa* - SIMION potential array of simple three-element lens
  • emittance.lua - SIMION user program associated with emittance.iob

Usage

The demo can be run as follows.

  • 1. View emittance.iob. (Click View on main screen and select the emittance.iob file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Fly particles. (Click Fly'm button.). Emittance calculation can be viewed in Log tab.
