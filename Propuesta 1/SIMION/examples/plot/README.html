SIMION Plotting Examples

Purpose

These examples demonstrate how to create graphical plots from SIMION user programs. Doing so can be useful for things like displaying the results of a simulation in the form of phase plots, surface plots, histograms, etc. The plotting is done in conjunction with either Microsoft Excel or gnuplot programs, and support for other programs may be added in the future. A library is provided (plotlib.lua) having a consistent API across these programs, so in theory you can write code that is portable across all supported graphing programs.

It is certainly possible to use Excel or gnuplot without plotlib.lua. More direct interaction with these programs are demonstrated in Example(excel) and Example(gnuplot) respectively.

Installation

These examples assume you have Microsoft Excel (possibly requiring a recent version) and/or gnuplot installed. A copy of gnuplot is available via the "Check for Updates" button on the SIMION main screen.

If you want to use plotlib.lua in your own project, it's suggested to copy the files examples\plot\plotlib.lua, examples\excel\excellib.lua, and examples\gnuplot\gnuplotlib.lua into your project folder. You can omit excellib.lua or gnuplotlib.lua if you don't need Excel or gnuplot support respectively. It also possible to omit plotlib.lua and import excellib.lua or gnuplotlib.lua into your program directly if you don't need the ability to switch between these programs at runtime.

plotlib.lua will automatically utilize either Excel or gnuplot, depending which is installed. You can force it to use one or the other by setting the global variable PLOT_LIBRARY to "excel" or "gnuplot" . A convenient way to do this is to enter a command like this in the SIMION command bar: 

PLOT_LIBRARY='gnuplot'

Files in this demo:

  • README.html - Description of example
  • phaseplot.iob - SIMION workbench demonstrating phase plotting
    • phaseplot.fly - SIMION particle definitions associated with phaseplot.iob.
    • phaseplot.pa* - SIMION potential array of simple three-element lens.
    • phaseplot.lua - SIMION user program associated with phaseplot.iob. This generates the phase plot using Excel. It uses excellib.lua.
  • tof_histogram.iob/lua - example computing and plotting histogram of TOF's.
  • surface_plot.lua - batch mode Lua program for plotting surface plot.
  • plotllib.lua - library that facilitates simplifies plotting 2D data with Excel or gnuplot. This is just a small wrapper around the examples\excel\excellib.lua and examples\gnuplot\gnuplotlib.lua libraries.

Usage (phaseplot.iob)

phaseplot.iob flies ions through a simple three-element lens and then plots a phase plot of the ion beam.

The phase plot demo can be run as follows:

  • 1. View phaseplot.iob. (Click View on main screen and select the phaseplot.iob file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Fly particles. (Click Fly'm button.). A phase plot is created in Excel and displayed on the screen.

Usage (surface_plot.lua)

Click "Lua Lua Program" on the SIMION main screen and select the "surface_plot.lua" file. A 2D surface plot will display.

Usage tof_histogram.iob

tof_histogram.iob can be run by loading it and clicking Fly'm.
