Chemical Reaction Simulation (RS) Example for SIMION

Overview

This example, Reaction Simulation (RS), utilizes the Lua user programming interface of SIMION to simulate the chemical reaction dynamics of charged particles under atmospheric pressure (AP) conditions, which is a common problem in the design and investigation of modern ion analytical devices like ionization sources for mass spectrometry or ion mobility spectrometers. It optionally interfaces with the SIMION SDS collision model.

Most details can be found in the included documentation (rs_simion_documentation.pdf) and on http://rs.ipams.uni-wuppertal.de .

Files in this Example

  • README.html - description of this example
  • simion_rs_documentation.pdf - more detailed documentation
  • RS.conf - reaction configuration file
  • RS*.lua - RS model implementation
  • test.iob - simple example workbench using RS model. See test.lua for details.
  • COPYING - software license referenced from code files

This example also requires these files from the other examples:

  • examples\collision_sds\collision_sds.lua (including m_defs.dat, mbmr.dat, arraylib.lua, and textfilelib.lua)
  • examples\contour\contourlib81.lua (optionally for gas flow plotting)

See Also

  • Example: SDS -- collision model used by RS.
  • Example: Contour -- library for plotting vector fields, optionally used by this example
