SIMION Particle Count / % Transmission Calculation Example

Purpose

This example demonstrates how to apply a short workbench user program to do things like count the number of particles hitting a particular electrode, calculate the percent of particles hitting the electrode (% transmission), and printing particle starting points vs. particle ending points.

It's possible to do similar things without a user program. Just use SIMION's Data Recording features (View screen > Particles tab > Data Recording) to record the appropriate data. For example, in Data Recording, set What Data to Record of Ion number, Events, and Position; When to Record to Ion's Splat or Crossing Plane X/Y/Z; and Format of Delimited and Output file to "output.csv". This can be loaded in Excel or a similar program for data analysis. However, it's convenient to instead do this calculation all inside SIMION during the particle flying via a workbench user program.

Files in this Demo:

  • README.html - Description of example
  • count.iob - workbench
  • count.fly2 - initial particle definitions (circular beam)
  • count.lua - workbench user program to count particles and print percent transmission.
  • lens.pa# - simple geometry of 2D cylindrical lens, with aperture and anode (used by count.iob).

How to run demo:

  • 1. View count.iob. (Click View on main screen and select the count.iob file.) If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 2. Click Fly'm.. See the Log window for transmission data. It will look like this: 
    ...
splat:	y_begin=	1.5465799692617	y_end=	-2.0245979186377
splat:	y_begin=	14.731130755129	y_end=	15.32090212714
num_hits=	21
num_particles=	200
efficiency (%)=	10.5
  • 3. Examine the user program to see how this is implemented. Click User Program on the Particles tab.
