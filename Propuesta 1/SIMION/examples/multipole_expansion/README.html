SIMION Multipole Expansion Example

Purpose

This examples demonstrates how to take electrostatic potentials (defined analytically or defined in a PA refined by SIMION) and calculate the multipole components from it.

Compatibility Notes

This example uses SIMION 8.1 features.

Files in this folder

  • calculate.lua - Lua batch mode program to perform multipole expansion on each PA. If the PAs don't exist, it creates them from corresponding GEM files and refines them. This is the program you run.
  • multipole_expansion.lua - Lua module to determine multipole expansion from potentials using matrix methods to solve a system of linear equations. Used by calculate.lua.
  • multipole_expansion_paper.lua - Lua module to determine multipole expansion from potentials via Bessel functions as discussed in paper. WARNING: This currently doesn't work since the formula in the paper appears incorrect. Use multipole_expansion.lua instead.
  • matrix.lua - Lua module for matrix computation. This is based on version 0.2.4 from http://lua-users.org/wiki/SimpleMatrix (minus support for complex numbers).
  • trap_truncated3.gem - hyperbolic electrodes truncated at 3 times ring radius
  • trap_truncated3_mirror.gem - same as trap-truncated3 but with mirroring enabled
  • trap_stretched.gem - same as trap-truncated3 but cap position stretched 10.8%
  • trap_ringslot.gem - same as trap-truncated but with ring slots
  • trap_endholes.gem - same as trap-truncated but with end cap holes
  • mathcad/* - these are mathcad files that generally duplicate the results in multipole_expansion.lua (matrix.mcd) and multipole_expansion_paper.lua (bessel.mcd). These files are from an old version of Mathcad, so you probably can't use them, but you can view the corresponding PDFs instead.

Usage

Run the "run.lua" batch mode program (click "Run Lua program" on the main sceren and select that file). The multipole coefficents for each PA will display in the Log window.
