SIMION™ 8.1

SIMION RF Examples

Purpose

These examples demonstrate multiple approches in generating a simple RF field between parallel plates via user programming.

Description

There are three examples. All are functionally identical but have different implementations:

  • elect.iob - This contains a fast adjust potential array. The user program contains a fast_adjust segment that defines the time-dependent potentials on the two electrodes in the fast adjust potential array. This is the most typical method of generating RF potentials.
  • factor.iob - This contains a basic potential array defining a static field between parallel plates. The user program contains an efield_adjust segment that multiplies the field and potentials in that PA by a time-dependent factor sin(ion_time_of_flight * omega).
  • area.iob - This contains a basic potential array. The user program contains an efield_adjust segment that defines the field and potentials inside a PA according to an analytical equation. The potential array itself could even be empty since the field is defined entirely from an analytical equation.

These should serve as simple examples of how things work and provide some options.

Files in this demo:

  • README.html - Description of example
  • elect.iob, factor.iob, and area.iob - SIMION workbenches for the three examples.
  • elect.fly, factor.fly, and area.fly - SIMION particle definitions for the three IOB files respectively.
  • elect2.fly - SIMION particle definitions - alternate with varying masses.
  • elect.pa*, factor.pa*, and area.pa* - SIMION potential array for each of the three examples respectively.
  • elect.lua, factor.lua, and area.lua - SIMION user programs for each of the three examples respectively.

Usage

How to run demos:

  • 1. View desired IOB. (Click View on main screen and select the desired IOB file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Set trajectory quality to zero. (In Particles tab, set TQual to 0.)
  • 4. Fly ions. (Click Fly'm button.)
  • 5. Cut off top electrode in view (with 3D zoom) to see isometric views. (See details on 2D/3D zooming/cutting options in printed SIMION manual. Press 3D Iso button.)

More:

  • 1. Fly ions Grouped in Rerun mode. (In Particles tab, enable Grouped and Rerun. Click Fly'm button.)
  • 2. Change adjustable variables as ions fly. (In Variables tab, edit variable and press Enter.)

More on ELECT.IOB's special features:

  • 1. View ions flying as dots on PE surface view with elect.iob. The user programs undulate the PE surface so you can see why the ions move as they do - powerful.
  • 2. Vary PE surface update time step to see its effect (from Variables tab, change pe_update_each_usec variable as ions fly).
  • 3. Adjust dots flying speed (with slider under Dots in Particles tab). You can adjust flying speed and surface update time step for smoothest view.
  • 4. Note: Increasing GStep from 2 to 8 speeds up surface redrawing. (PE Contours tab, GStep)
  • 5. Fly elect2.fly to see effects of varying mass. (In Particles tab, click Define..., click Load, select elect2.fly file, and press OK.)

Historical Note

  • factor.iob simulates a SIMION 4.0 .FAC program with an efield_adjust seg
  • area.iob simulates a SIMION 4.0 .AR? program with an efield_adjust seg
  • elect.iob simulates a SIMION 4.0 .ELE program with an fast_adjust seg
