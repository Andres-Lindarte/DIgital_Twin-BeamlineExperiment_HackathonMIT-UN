PA2Text Example

Purpose

This is a simple example of reading a PA file using the SL Libraries. This example requires some familiarity with C++, Perl, Python, or Lua programming languages.

Description

This example is implemented in three programming languages:

  • cpp - C++ implementation
  • perl - Perl implementation
  • python - Python implementation
  • lua - Lua implementation (SIMION 8.1)

Usage

CPP version

Use: To use, compile the pa2text.cpp program into pa2text.exe and run the pa2text.bat program, which passes the example simple.pa# to pa2text.exe. Build files are provided for Microsoft Visual C++ 6.0 and GNU g++ C++ compilers. If you use another compatible c++ compiler, you just have to make sure that you add the SL Libraries for C++ (lib\cpp) to your include seach path and make sure pa.cpp gets compiled into the binary).

Perl version

Use: To use, just run the field.bat program. This will execute the Perl interpreter and point it to the location of the SL Libraries for Perl (lib\perl).

Python version

Use: To use, set the "PATH" variable in field.bat to point to the directory in which python.exe is installed on your system. Then run field.bat.

Lua version

Use: This requires SIMON 8.1. Click "Run Lua Program" from the SIMION main screen and select pa2text.lua.
