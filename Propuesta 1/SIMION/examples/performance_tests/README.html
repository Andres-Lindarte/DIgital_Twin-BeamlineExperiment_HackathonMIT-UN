SIMION™ 8.1

SIMION Performance Tests

Purpose

The performance tests are designed to test the simulation accuracy of SIMION for various simulation problems.

Description

The problems chosen have analytical solutions. Thus the expected values are known to the precision of physical constants (8 - 9 places). SIMION assumes the values of the physical constants from the 74th edition of the handbook of chemistry and physics (1993-1994).

Each simulation displays both the expected (analytical) and SIMION computed values for the measure(s) of interest. In certain cases, increasing the trajectory quality from 3 (its default) to 103 or above improves the accuracy of the SIMION computed values. It is suggested that you adjust trajectory quality to see its effect on the various tests.

IMPORTANT! Read the README.TXT file in each directory to learn how to setup and conduct the simulations contained therein.

A simulation contains a collection of files of the same name. Before a simulation can be run its .GEM files (if any) must be processed to create .pa# files of the same name. Then any .pa# files must be refined. It is suggested that all .pa# files be refined to 1.0e-7 convergence. However, you may wish to use the default refining 5.0e-3 convergence to compare the effects of the various levels of refining.

To run a canned simulation test (after creating and refining the required files), load the simulation's .iob file into view and click the Fly'm button.

Each directory has a README.txt file to explain its simulations. It is suggested that you work through the simulation tests in order (e.g. 1-8). This is the logical order in increasing complexity of the tests.
