The tests performed are simple ke to speed back to ke calculations.  The
ions are flying in a field free volume.  Both non-relativistic and
relativistic conversion tests are performed.

The objective of these tests is to test SIMION's conversion of mass and KE
into speed, and the reverse conversion back into KE.  Remember SIMION
assumes all masses are rest masses (0 velocity masses).

No array preparation is required. These tests can be run now.
