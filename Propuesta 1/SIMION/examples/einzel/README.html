SIMION Einzel Lens Example

Purpose

The files in this directory provide an example of a simple einzel lens.[1]

Files in this demo:

  • README.html - Description of example
  • einzel.iob - SIMION workbench
  • einzel.fly - SIMION particle definitions associated with einzel.iob
  • einzel.pa* - SIMION potential array of simple three-element lens
  • einzel.gem - geometry file optionally used to create einzel.pa#. Einzel.pa# can be easily created in Modify without GEM files though. einzel_cylinder.gem and einzel_param.gem illustrate two alternate ways to define the GEM file.

Usage

How to run demo:

  • 1. View einzel.iob. (Click View on main screen and select the einzel.iob file.) You will not be asked to refine it since this example comes pre-refined.
  • 2. Set trajectory quality to 0. (Set TQual in Particles tab to 0.)
  • 3. Fly ions Grouped in Rerun mode. (Enable Grouped and Rerun options on Particles tab and then click Fly'm.) Notice point of focus. Ions will continually refly, so continue to next step.
  • 4. Change electrode 2 to 150 volts and see the change in focus point. (In PAs tab, click Fast Adjust Voltages, change electrode 2 to 150 volts, and press Fast Adjust.)
  • 5. Switch to PE view and enable Dots. (With XY button down, press down PE button. In Particles tab, enable Dots.) Now you can see why the ions focus.
  • 6. Exit PE view and stop flying ions. (Unpress PE button. Unpress Fly'm button.)
  • 7. Enable Beam Repulsion. (In Particles tab, set Repulsion to Beam.)
  • 8. Fly the ions, and increase the beam current to 1.0e-7 amps. (Click Fly'm button. In Particles tab, under Beam, type 1.0e-7 and press Enter.) Notice the effect this has on ion trajectories.

Note: This is a good demo to use to learn about 3D zooming and other basic SIMION features. Read your manual, spend some time and enjoy!

Background

See Chapter 2 "SIMION Basics" of the printed SIMION manual for a description of how to create an example similar to this from scratch. The Tour of the SIMION Demo also contains a good look at the Einzel example.

Geometry Definition

einzel.pa# is a simple 2D cylindrical geometry with three electrodes. It is easily drawn using SIMION Modify using methods similar to that described in Chapter 2 "SIMION Basics" of the printed SIMION manual. If you prefer, einzel.pa# may alternately be created from a GEM file. Three GEM files are provided in this directory illustrating different approaches you may take for defining the GEM file (einzel.gem is the simplest). There is also an STL file (two_cylinder.stl) in Example(geometry) that somewhat resembles this geometry.
