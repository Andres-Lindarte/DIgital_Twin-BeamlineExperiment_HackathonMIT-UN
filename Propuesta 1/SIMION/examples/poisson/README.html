SIMION™ 8.1

SIMION Poisson Solving Examples

Purpose

The examples in this folder demonstrate the ability of SIMION 8.1's Refine function to solve the Poisson equation. The main use of this feature is to solve electric fields while taking into account space-charge density distributions that are either known or determined by particle tracing. The later includes situations where particle trajectories (space-charge) and fields are mutually dependent, requiring iteratively solving until a self-consistent solution is reached (i.e. particle-in-cell, "PIC", methods). The Poisson equation is also useful for other things like solving vector magnetic potential (which involves current densities rather than charge densities), the stationary heat equation, and other things.

Compatibility

This example uses new SIMION 8.1 features, such as the The Poisson Solver and the Lua PA API (simion.pas).

Description

For a background of SIMION's new Poisson solver, see The Poisson Solver in SIMION. It is recommended that you review that before using these examples.

Note: the Poisson solver is an advanced feature. It currently requires Lua user programming to effectively make use of it, but it can be very powerful and flexible.

Files in this demo:

  • README.html - Description of example.
  • [Example: poisson\static_charge] - Poisson solving examples utilizing known static distributions of charge. This does not involve particle tracing. See also "pipe_with_charge".
  • [Example: poisson\pierce_gun] - Pierce gun examples - a simple particle gun with space-charge effects. Use PIC in both current and charge modes.
  • [Example: poisson\pipe_with_charge] - calculation of potentials given a cylinder of static charge inside a conductive pipe. Includes Poisson solving with image charge effects.
  • [Example: poisson\pipe_with_beam] - similar to "pipe_with_charge" but the charge is formed by tracing a particle beam. Uses PIC, like the "pierce_gun" example, in current mode. Analysis of image charge effects is emphasized.
  • [Example: poisson\octupole_sc] - RF octupole with space-charge effects. This demonstrates time-dependent (RF) fields with the Poisson solver.
  • [Example: magnetic_potential] - using the Poisson solver to solve magnetic vector potentials
  • piclib.lua - particle-in-cell (PIC) utility code for ray tracing. Used by examples like the Pierce gun.
  • palib.lua - PA manipulation utility functions. Used by a number of examples.

Usage - static_charge examples

These examples demonstrate computing a field when the charge density distribution is already known. See Example: poisson\static_charge for details.

Usage - pipe_with_charge examples

This is similar to the static_charge examples. Specifically, it calculates the electric field due to a cylinder of charge inside a conductive pipe. It also analyzes the image charge effect that occurs on the pipe due to the cylinder of charge, which the Poisson solver handles but the charge repulsion methods do not (unless image charge is added explicitly). See Example: poisson\pipe_with_charge for details.

Usage - pipe_with_beam examples

This extends pipe_with_charge in that the charge is not static but is formed by tracing a particle beam. Fields and particle trajectories are solved by a self-consistent method ("piclib" current mode). See Example: poisson\pipe_with_beam for details.

Usage - pierce_gun examples

These examples (like pipe_with_beam) demonstrate simulating a system when the field and particle trajectories are mutually coupled. The space-charge from the particle trajectories is a charge density distribution that perturbs the electrode surface charges (and hence the electric field) that in turn perturbs the particle trajectories that in turn perturbs the field, ad infinitum. Both "current" (time-independent space-charge) and "charge" (time-dependent space-charge) handling modes are demonstrated. An example including ion-gas collision effects is also provided. See Example: poisson\pierce_gun.

Usage - octupole_sc examples

This example demonstrates an RF octupole ion guide with space-charge effects simulated with the Poisson solver. This is an example of time-dependent (RF) fields with the Poisson solver. Charge repulsion methods are also tested for comparison. See Example: poisson\octupole_sc.

Usage - magnetic_potential

These examples use the Poisson solver to determine magnetic fields given three magnetic PA's (for the x, y, and z components of magnetic vector potentials) and three charge PA's (for the x, y, and z components of the current density vector). This is useful for non-zero current density situations. Magnetic Potential has some background notes on using the Poisson solver for solving magnetic vector potentials. See the Example: magnetic_potential.

See Also

  • Example: Dielectric includes a few static field calculation examples involving dielectrics and space-charge simultaneously (including a charged floating conductor).
  • The Poisson Solver in SIMION - a description of SIMION's Poisson solver.
  • Example: Charge Repulsion demonstrates the SIMION 7.0 and 8.0 "charge repulsion" feature that approximates space-charge effects assuming only ion-ion repulsions (ignoring interactions with electrode surface charges, which require use of the Poisson solver). This is an alternate approach to space-charge handling that is in general not as exact may still be useful or convenient in certain situations.
  • simion.pas - This library is frequently used by these examples to manipulate and refine potential arrays.
