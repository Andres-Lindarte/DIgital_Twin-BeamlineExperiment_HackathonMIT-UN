SIMION Static Charge Poisson Field Solving Examples

Purpose

This demonstrates using the Poisson solver to calculate fields given electrodes and known distributions of static charge. These examples do not involve particle tracing.

Files in this demo:

  • sphere_simple.lua - calculates electric field of uniform sphere of charge inside a larger conductive sphere. batch mode program. 2D cylindrical (spherically symmetric).
    • sphere_simple.gem - electric field PA definition
    • sphere_simple-charge.gem - space-charge PA definition
  • sinusoidal_pla.lua - calculates electric field of sinusoidal charge density. batch mode program. 2D and 3D planar.
  • sheets_2dcyl.lua - calculates electric field of 2D cylindrical sheets of charge. batch mode program. 2D cylindrical.

Theory of sphere_simple.lua

The "sphere_simple.lua" example simulates a sphere of radius R1 (mm) of uniform charge density ρ (C/mm3) inside a larger spherical conductor (radius R2) at ground potential. The field E in this example has a known theoretical solution, which due to the spherical symmetry, we can determine simply by Gauss' law:

E = (ρ/(3 ε0)) r   (for |r| <= R1)
E = (ρ/(3 ε0)) R13 / r2   (for |r| >= R1)

The corresponding potentials φ are therefore expected to be as follows, which SIMION confirms:

φ = (ρ/(3 ε0)) [ -(1/2)r2 + (3/2)R12 - R13/R2 ]   (for |r| <= R1)
φ = (ρ/(3 ε0)) [ R13/r - R13/R2 ]   (for |r| >= R1)

Theory of sinusoidal_pla.lua

The "sinusoidal_pla.lua" example solves a conductive box (in 2D planar or 3D planar symmetry) with space-charge densities of the form

ρ(x,y,z) = ρmax * sin(i π x/L) * sin(j π y/M) * cos(k π z/N)

This system has a theoretical solution. See the comments in sinusoidal_pla.lua for details. This example is a bit more complicated but uses utility functions in "palib.lua" to help setup the system.

Theory of sheets_2dcyl.lua

The "sheets_2dcyl.lua" example solves the field from a system consisting of any number of concentric cylindrical sheets of charge of various charge densities. The space-charge density is therefore an arbitrary function of only the radial distance r (i.e. with no dependence in the axial direction). This system has a theoretical solution. See the comments in sheets_2dcyl.lua for details.

How to run this demo:

  • 1. Run the sphere_simple.lua in batch mode program. (Click Run Lua Program on the main screen and select the Lua file.) The program will build the arrays, refine them, and display a comparison of actual calculated potentials to theoretical ones. You can examine the sphere_simple.pa in the View screen, such as to plot PE or contours.
  • 2. Run the sinusoidal_pla.lua or sheets_2dcyl.lua batch mode program. (Click Run Lua Program on the main screen and select the Lua file.) The program will build the arrays, refine them, and report the relative error of the SIMION computed potentials against theoretical expectations (approximately 0.001).

Examining the source code is highly instructive and important to understanding these examples.
