SIMION Spherical Test Surface Example

Purpose

This example demonstrates how to record data when particles cross a non-planar surface (e.g. a sphere). It utilizes a Lua workbench user program (surf.lua) attached to the workbench. Examine the surf.lua file in a text editor for details.

Note that there may be easier ways to do this type of thing depending on the circumstances:

  • If you only need to record data when particles cross an X, Y, or Z plane, then you can instead use Data Recording "When to Record > Crossing Plane".
  • If you only need to record data when particles cross one or more arbitrary planes (not necessary orthogonal to the X, Y, and Z axes), then you may instead use the similar approach in the "test_plane" example.

Files in this demo:

  • README.html - Description of example
  • surf.iob - SIMION workbench
  • surf.fly2 - SIMION particle definitions associated with surf.iob. Conical beam shape.
  • surf.pa - Empty SIMION potential array.
  • surf.lua - SIMION workbench user program associated with surf.iob.

Usage

To run this example, view the surf.iob and click Fly'm. Note that whenever a particle crossing the defined spherical surface (radius=10.5 mm and center at origin) a red dot is displayed on the graphics window and a time-of-flight is displayed in the Log window (5.25 micrseconds is expected for all particles).
