SIMION Ion Mobility Example

Purpose

This example models an ion mobility drift cell.

Description

This model is based on the one in the paper by [Gillig et al].

Gillig et al modeled the background gas effect using a hard-sphere collision model, probably related to but simpler than Exampe: HS1 Hard-Sphere Collision Model. The present example supports both the simple "Stokes' law" drag model from Example: Drag as well as Example: HS1 Hard-Sphere Collision Model. The HS1 model can provide more realistic effects but gets more computationally intensive at higher pressures. Note that the drag model does not model diffusion: ions with zero initial KE under a zero field should still move away from each other due to thermal collisions with the background gas, which does occur in HS1 but not when using the plain Stokes' law.

Ion Mobility Constant

The ion mobility constant K is defined as

vd = K E,

where vd is the (terminal) drag velocity of the ion when subjected to an electric field E. K can be expressed in terms of the reduced ion mobility constant:

K0 = K (P/P0) (T0/T),

where P0 = 760 Torr and T0 = 273.15 K as noted in [1].

Notes on the Stokes' Law (Drag) Model and Mobility

The drag model assumes a Stokes' Law condition, in which the additional negative acceleration (as) on the ion as due to the background gas is proportional to the ion velocity v:

as = - λ v.

λ is in units of 1/microseconds in the drag model.

With Stokes' Law effects, terminal velocity (vd) occurs when the total acceleration is zero: as + af = 0, where af is the acceleration due to the field E. as = -λ vd by Stokes' Law. af is simply af = F / m = q E / m = (q/m) E where q/m is the mass-to-charge ratio of the ion. Therefore, 0 = as + af = -λ vd + (q/m) E, which solves to vd = (q/m) E (1/λ). Plugging vd into the ion mobility equation vd = K E above gives (q/m) E (1/λ) = K E. That is,

λ = (q/m) (1/K).

This λ can be defined in the Drag model.

In λ = (q/m) (1/K), we might prefer using units of e for q, u for m, and (cm/sec)/(V/cm) = cm2 V-1 s-1 for K. λ is in units of microsecond-1. This requires a unit conversion factor of (1.60217653(14)E-19 C e-1) (1.66053886E-27 kg u-1)-1 (100 cm m-1)2 (10-6 sec microseconds-1) = 9.648533798E+5 C e-1 kg-1 u cm2 m-2 sec microseconds-1 = 9.648533798E+5 V-1 s-1 e-1 u cm2 microseconds-1.

Gillig et al give a value K0 = 4.31 and use q/m=720, pressure = 1 Torr, and temperature = 298 Kelvin. (Units on K0 are presumably cm2 V-1 sec-1 at P0 = 760 Torr and T0 = 273.15 K, which is consistent as noted in the collision cross section calculation below.) Therefore, by the above equations, K = K0 (P0/P) (T/T0) = 3.57(4)E+3 cm2 V-1 sec-1, and λ = (q/m) (1/K) = (9.648533798E+5 V-1 s-1 e-1 u cm2 microseconds-1) (1 e / 720 u) (3.57(4)E+3 cm2 V-1 sec-1)-1 = 0.375 microseconds-1. This value of λ is defined in the user program.

Notes on the Hard Sphere (HS1) Model and Mobility

[1] gives a relationship between the mobility constant K, and Ω, which is approximately equal to the collision cross-section:

Ω = (3/16) q N0-1 (2 π / (μ k T))1/2 K-1

where N0 is the buffer gas number density, μ is the reduced mass (reciprocal of the sums of the reciprocal masses of the colliding particles), and T is the effective temperature.

Using the ideal gas law in terms of the Boltzmann constant k, we find N0 = (N/V) = P k-1 T-1 = (1 Torr) (133.322 Pa Torr-1) (1.3806505(24)E-23 J K-1)-1 (298 K)-1 = 3.24(0)E+22 m-3. Further, μ = (mHe-1 + mC60-1)-1 = ((4.002602(2)u)-1 + (720u)-1)-1 = 3.981 u. Therefore, Ω = (3/16) q N0-1 (2 π / (μ k T))1/2 K-1 = (3/16) (1.60217646E-19 C) (3.24(0)E+22 m-3)-1 (2 π / (3.981u * 1.66053886E-27 kg/u * 1.3806505(24)E-23 J/K * 298 K))1/2 (3.57(4)E+3 cm2)-1 (m/100cm)2 = 1.25E-18 m2. This value of Ω is defined in the HS1 program (_sigma_m2 variable). Note: This Ω is fairly consistent with a C60 outer diameter of 10.18 Angstrom[2] and a relatively small He diameter.

Geometry

Similar (but not identical) to in Gillig et al, the ring electrodes are of 64 mm O.D., 7.5 mm I.D., 6.5 mm width, and 6.5 mm spacing. Field is approx 50 V/cm. (Gillig notes He supports maximum voltage of 68 V/cm at 1 Torr.)

Usage

How to run demo:

  • 1. View ims.iob. (Click View on main screen and select the ims.iob file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Select trajectory quality = 0 and disable Rerun. (In Particles tab, set TQual to 0 and disable Rerun.)
  • 4. Fly ions. (Click Fly'm.) Note how the ions maintained focused along the axis. Use asymptotic zooming (in Display tab, uncheck "= Zoom" and then 2D zoom on an area) to zoom in on the ion beam (expanding Y directory to see small oscillations in Y).
  • 5. Enable the hard-sphere model, reload the workbench, and refly the ions. In the bottom SIMION command-bar enter mode="hs1", reload the workbench and click Fly'm. Note how the ions maintained focused along the axis (but behaves somewhat differently than before). You can revert to the drag model by entering mode="hs1" into the SIMION command-bar and reloading again.

Note: The workbench program ims.lua merely calls either drag.lua or collision_hs1.lua in the same directory and sets parameters appropriately. If you need only one of the models, you could simplify the code a bit by removing the other model.

Note: to view the linear_damping variable, check "Show All" on the Variables tab.

Possibly try turning off the background gas (set _preesure_pa to 0 in HS1 or set linear_damping to 0 in Drag.) Note that there is no longer axial confinement.
