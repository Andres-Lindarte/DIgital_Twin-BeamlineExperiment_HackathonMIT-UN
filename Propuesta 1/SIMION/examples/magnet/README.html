SIMION Magnet Examples

Purpose

The files in this directory provide examples of modeling magnetic effects.

Description

Examples included are

  • mag90.iob - Demostrates z focusing effects of offset beam entry
  • magnet1.iob - Demonstrates combined electrostatic and magnetic focusing effects on ions of opposite charge

Files in this demo:

  • README.html - Description of examples
  • Mag90 - simple 3D magnetic sector
    • mag90.iob - SIMION workbench - 90 degree magnet
    • mag90.fly - SIMION particle definitions associated with mag90.iob
    • mag90.pa* - SIMION potential array for mag90.iob
    • mag90.gem - SIMION geometry file, which can optionally be used to generate mag90.pa#
  • Magnet1 - combined magnetic-electrostatic lens
    • magnet1.iob - SIMION workbench - combined magnetic-electrostatic focusing lens
    • magnet1.fly - SIMION particle definitions associated with magnet1.iob
    • magnet1.pa* - SIMION potential array - magnetic component
    • magnet1.gem - SIMION geometry file, which can optionally be used to generate magnet1.pa#
    • elect1.pa* - SIMION potential array - electrostatic component
    • elect1.gem - SIMION geometry file, which can optionally be used to generate elect1.pa#

Usage

How to run z focusing demo:

  • 1. View mag90.iob. (Click View on main screen and select the mag90.iob file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Set trajectory quality to 0. (In Particles tab, set TQual to 0.)
  • 4. Fly ions. (Click Fly'm button.) Note the z focusing effects on the two beams of ions.
  • 5. Cut the magnet open and look inside. (See description of 2D/3D zooming options in printed SIMION manual, or skip this.)
  • 6. Fly ions Grouped as Dots in Rerun mode. (In Particles tab, enable Grouped, Dots, and Rerun. Click Fly'm.)
  • 7. Experiment.

How to run combined electrostatic and magnetic focusing demo:

Note: Both the electrostatic and magnetic PAs are superimposed to create combined electrostatic and magnetic effects.

  • 1. View magnet1.iob. (Click View on main screen and select the magnet1.iob file.)
  • 2. If this is your first time using the example, you will be asked to refine (confirm that and press Refine).
  • 3. Hide display of electrostatic PAs. (In Display tab, unpress the Ele button.) This speeds and improves the Iso views.
  • 4. Set trajectory quality to 0. (In Particles tab, set TQual to 0.)
  • 5. Fly ions in Rerun mode, Grouped, as Dots and at max speed. (In Particles tab, enable Rerun, Grouped, and Dots. Move slider below Dots to right for max speed. Click Fly'm.)
  • 6. Switch to ZY view. Note the ion motions.
  • 7. From ZY view, make a 3D zoom, leaving the lens hole within the upper left corner of the new volume (left and top lens material removed). (While in ZY view, drag rectangle from upper-left of hole to lower-right of electrode. Click +Z3D button. Then click 3D Iso button for 3D isomeric view. Increase Display Qual to 8. Optionally 2D zoom by dragging a rectangle around the electrode and pressing Z2D button (or right mouse clicking). See printed SIMION manual for 2D/3D zooming details.)
  • 8. Switch to 3D isometic view if not already. (Press 3D Iso button). Watch the ions.

Note:

  • Ions of opposite polarity are twisted in opposite directions in the magnetic gap.
  • Ions of opposite polarity are accelerated differently accross the the electrostatic gap (same as magnetic gap).

Extra thrills: Fast adjust the magnetic field to 200 -- see the effect it has on the ion trajectories.

Comments

Scalar potential: This example assumes that the magnetic scalar potential is constant over the pole faces, or at least approximately so on those sections of the surface significantly affecting fields in regions where the ion trajectories go through. See Example: magnetic_potential (mag_2cylinder_2dc.iob) for a perhaps more realistic simulation that does not make this assumption.

Geometries: The potential array files in this example were probably originally generated merely used SIMION Modify. They are simple 2D geometries that are readily drawn in Modify. GEM have also been included though in case you want to customize these examples using GEM files.
