This folder contains 32-bit versions of SIMION
binary files.  Normally you should use the 64-bit
version of SIMION in the main directory instead,
but the 32-bit version is here if you want it.

Note that 64-bit executables cannot load 32-bit DLLs.
So, if Lua needs to load 32-bit DLLs, you should use
the 32-bit SIMION.
